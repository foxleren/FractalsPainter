Немного о функционале:
* Доступно 5 фракталов для отрисовки. Кол-во итераций ограничено для
	сохранения максимальной отзывчивости интерфейса.
* Все параметры задаются с помощью слайдеров при перетаскивании пупырки, у которых 
	показываются подсказки о текущем значении величины.
* Изначально интерфейс рекомендуется использовать при открытом на полный экран приложении.
	Но он адаптивен для любого допустимого разрешения. Канвас автоматически обновляется 
	при изменении разрешения окна (будет отображаться полоса прогрузки с таймером,
	обрати на нее внимание, не зря же я ее делал хахахаха).
* Параметры стоит задавать после выбора фрактала, тк при новом выборе все параметры сбрасываются.
   Если не выбрать фрактал и нажать кнопку отрисовки, то будет сообщение.
* По умолчанию к фигурам применяются стартовые цвета из палитры. Если ты хочешь изменить цвет, то
	сначала выбери цвет на палитре, затем нажми кнопку для отрисовки.
* Если фрактал уже построен, то можно менять параметры в реальном времени и наблюдать за изменениями.
	(для пифагорова дерева и множества Кантора). Возможность менять итерацию в реальном времени доступна
	для всех фракталов после их первой отрисовки.

Системные требования:
 * Ryzen 4600H
 * 16Gb DDR4
 * Windows 11
